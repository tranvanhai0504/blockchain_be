API Usage Guide

Product Endpoint

        GET /api/v1/product/all/:id              // Get all products :id = userWalletId
        GET /api/v1/product/:id                 // Get product by ID
        POST /api/v1/product/add                // Add a new product
            Request Body example:
            {
                "name": "Product Name",
                "price": 20.99,
                "description": "Product Description"
                "userWalletId" : '1xxxx223123'
                "media": "-----" 
            }

        PUT /api/v1/product/edit/:id         // Edit a product by ID
            Request Body example:
            {
                "name": "Updated Product Name",
                "price": 25.99,
                "description": "Updated Product Description",
                "media": "-----" ,
                 "userWalletId" : '1xxxx223123'
            }

User Endpoint

        GET /api/v1/user                 // Retrieve all users
        POST /api/v1/user                // Create a new user

QR Endpoint

        GET /api/v1/qr                   // Retrieve QR code
        POST /api/v1/qr                  // Create a new QR code
